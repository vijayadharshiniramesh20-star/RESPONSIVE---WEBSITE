# Responsive Website

## 📌 Project Overview
This project is a simple responsive website built using HTML and CSS. It adjusts automatically to different screen sizes, providing a user-friendly experience on desktop, tablet, and mobile devices.

## 🚀 Features
- Responsive design
- Navigation bar
- Header section
- Information cards
- Footer section
- Mobile-friendly layout

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📂 Project Structure
```
Responsive-Website/
│── index.html
│── style.css
│── README.md
```

## ▶️ How to Run
1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` in your web browser or use the Live Server extension.

## 📸 Output
- Responsive desktop view
- Responsive mobile view
